This is a boilerplate for a REST-API by using Go.

# How to build
```
docker build -t boilerplate_go_rest:latest .
```

# How to run

```
docker run -p 3000:3000 boilerplate_go_rest:latest
```

Now, you can go to http://127.0.0.1:3000/swagger/index.html
